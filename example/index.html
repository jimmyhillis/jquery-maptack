MapTack

A simple JavaScript geolocation form widget

This tool allows you to quickly search and geo-locate addresses (or any point) into latitude and longitude values within an HTML form. Often useful for mapping locations or points for a public facing map.

This plugin required jQuery and makes use of the Google Maps APIv3 for plotting and geo-locating addresses.
